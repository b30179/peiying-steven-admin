# Steven S1 标书／文书模块演示脚本 v1

- 日期：2026-07-17
- 演示范围：本机独立脱敏 Demo
- 建议时长：5–8 分钟
- HTTPS 入口：`https://localhost:15443`

## 1. 演示前检查

1. 运行 `D:\ZM\AI工坊\培英\04-开发实现\scripts\start_steven_demo.ps1`。
2. 确认浏览器使用 `https://localhost:15443/login`，不要使用内部 HTTP 端口登录。
3. 确认 `/health` 为：
   - `auth_mode=session`
   - `persistence.mode=postgresql`
   - `tender_store=postgresql`
   - `demo_seed_enabled=false`
   - `ocr_enabled=false`
   - `ai_enabled=false`
4. 只使用项目负责人保管的受控 Demo 账号；演示材料必须保持脱敏。

## 2. Steven 用户演示

### 步骤 1：登录

- 从 HTTPS 登录页登录。
- 说明 Session 使用 Secure/HttpOnly Cookie，前端不能传入角色或 actor。

### 步骤 2：打开 S1

- 进入 Dashboard。
- 点击“标书与报价单”或“新建标书”。
- 页面地址：`https://localhost:15443/dashboard/steven/tenders`。

### 步骤 3：新建脱敏文书

- 选择“脱敏采购／服务邀请文书”模板。
- 填写：
  - 脱敏标题与文书编号；
  - 脱敏事项名称；
  - 生成日期；
  - 截止日期；
  - HKD 预算上下限；
  - 演示地点；
  - 三个演示供应商；
  - 受控条款。
- 强调页面和 Word 均使用脱敏 Demo 内容。

### 步骤 4：展示规则阻断

按顺序展示一至两个阻断即可，完整规则由测试报告覆盖：

1. 将截止日期设为生成日期后不足 3 天，保存应被阻断。
2. 将预算下限设为负数或高于上限，保存应被阻断。
3. 输入大小写或全角形式等价的重复供应商，保存应被阻断。
4. 使用含未替换 `{{...}}` 的受控测试模板时，预览进入 `draft_error`，不能提交或正式导出。

### 步骤 5：修正并生成草稿

- 修正日期、预算、供应商和模板变量。
- 点击预览，确认规则错误清零。
- 下载 Word 草稿。
- 打开草稿，指出“草稿／未批准／仅供人工复核”标识。
- 人工修订受控条款后再次预览。

### 步骤 6：提交

- 点击提交审批。
- 展示状态变为“已提交”。
- 尝试使用提交人审批，系统应返回 403，证明自审被阻断。

## 3. 审批人演示

### 步骤 7：批准或退回

- 登出 Steven，使用独立审批人账号登录。
- 打开同一文书事项。
- 可先演示“退回但不填写意见”被阻断。
- 若演示退回：填写意见后退回，证明退回事项不能正式导出；由 Steven 修订后重新提交。
- 最终由审批人填写人工意见并批准。

审批人不得修改供应商、预算或文书正文；审批状态和 actor 均由服务端控制。

## 4. 正式 Word 与版本历史

### 步骤 8：导出 v1、v2、v3

- 返回具备导出权限的 Steven 账号。
- 对已批准事项连续执行三次正式导出。
- 确认生成 v1、v2、v3，文件名和 storage key 不同。
- 分别下载并打开三个 Word，确认均可读取。

### 步骤 9：查看历史

- 在详情页查看版本历史。
- 强调：
  - 历史版本不覆盖；
  - 版本号不回收或复用；
  - 文件有 size、SHA-256、创建人和状态元数据；
  - 文件发布与数据库登记采用可恢复 Saga。

## 5. 登出验证

### 步骤 10：退出登录

- 点击登出。
- 刷新受保护页面或使用旧会话访问，预期返回未认证并跳转登录。
- 说明 Session 已在 PostgreSQL 中撤销，不能靠保留旧 Cookie 继续访问。

## 6. 演示讲解边界

必须明确：

- 当前完成的是 S1 本机离线脱敏 Demo。
- Word 内容必须人工复核；系统不替代审批人。
- OCR/AI live 开关关闭，未调用 Azure或 DeepSeek。
- 未来 S1 OCR/AI 只能形成 `needs_review` 候选，不能直接写正式文书。
- S3 尚未实现。
- D2 在线 OCR/AI 尚未批准。
- 本演示不代表生产可用或可受控试运行。

`D:\ZM\AI工坊\培英\04-开发实现\scripts\stop_steven_demo.ps1`

停止脚本只停止本项目 Caddy、Next.js 和 FastAPI 进程，不删除 PostgreSQL 数据、账号、migration、审计或 Word 历史。
