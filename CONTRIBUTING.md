# 贡献指南

感谢您对 **某中學 Steven 行政 AI 助手** 项目的关注！

## 项目背景

本项目是为香港某中学行政老师打造的 AI 辅助办公系统，涵盖文檔智能 OCR、標書校对、采购比价、库存盘点等场景。

## 如何贡献

### 报告问题

- 使用 [GitHub Issues](https://github.com/b30179/mzx-steven-admin/issues) 提交 Bug 或功能建议
- 请描述：复现步骤、预期行为、实际行为、环境信息

### 提交代码

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "feat: 描述你的改动"`
4. 推送到分支：`git push origin feature/your-feature`
5. 提交 Pull Request

### 提交规范

遵循 [Conventional Commits](https://www.conventionalcommits.org/zh-hans/)：

- `feat:` 新功能
- `fix:` 修复 Bug
- `docs:` 文档更新
- `refactor:` 代码重构
- `test:` 测试相关

## 开发环境

- Python 3.11+
- FastAPI
- Next.js 16
- PostgreSQL 18
- PaddleOCR

## 行为准则

请保持专业、友善的交流氛围。
