# Steven 便携脱敏 Demo 使用手册

版本：v1（2026-07-18）

## 1. 适用范围

本交付包用于 Windows 10/11 x64 电脑上的本机离线演示，包含 FastAPI、Next.js、PostgreSQL 18、Caddy、PaddleOCR 模型与标准脱敏演示数据。系统只监听本机 `127.0.0.1`，浏览器唯一业务入口为 `https://localhost:15443`。

本包不是生产系统、不是受控试运行环境，也不预置可分发的真实 DeepSeek API Key。PaddleOCR 为真实本机离线识别；AI 结构化与校对默认使用明确标记的本地 Mock，也可由使用者在目标电脑上通过配置窗口填写 DeepSeek Endpoint、模型与 API Key 后显式启用在线模式。

## 2. 电脑要求

- Windows 10 或 Windows 11，64 位。
- 建议至少 16 GB 内存、15 GB 可用磁盘空间。
- 建议解压到较短、可写、非网络映射的本地路径，例如 `D:\StevenDemo`。中文路径可以使用；首次初始化或启动时，脚本会自动创建临时 ASCII 虚拟盘符供 PostgreSQL 使用，停止后自动释放。运行期间不要手动断开该虚拟盘。
- 不需要另行安装 Python、Node.js、PostgreSQL、Caddy 或 PaddleOCR。
- 首次初始化和 OCR 首次加载可能需要数分钟，请勿重复点击启动。

## 4. 首次安装与启动

1. 将 ZIP 完整解压到本机可写目录，建议使用较短路径；可以使用中文路径，但不要直接在压缩包内运行。
2. 双击 `首次初始化Steven便携Demo.cmd`。
3. 等待窗口显示初始化成功。脚本会：
   - 初始化包内 PostgreSQL；
   - 仅监听 `127.0.0.1:55432`；
   - 随机生成数据库密码并使用 Windows DPAPI 保护；
   - 恢复脱敏演示数据库；
   - 验证仅有 Steven、approve 两个账号；
   - 清空历史浏览器 Session。
4. 双击 `启动Steven便携Demo.cmd`。
5. 浏览器打开 `https://localhost:15443/login`。
6. 如果浏览器提示本地证书未受信任，可在首次启动后双击 `信任Steven本地HTTPS证书.cmd`，按 Windows 管理员确认完成本机证书信任，然后重新打开浏览器。


### 可选：配置 DeepSeek 在线 API

1. 先停止 Steven Demo，再双击根目录的 `配置Steven大模型API.cmd`。
2. 配置窗口默认填写：
   - Endpoint：`https://api.deepseek.com/v1`
   - 模型：`deepseek-chat`
3. 勾选“启用在线 DeepSeek”，在密码框中填写 API Key。Key 不会显示明文。
4. 可点击“测试连接”。测试只发送最小连通性提示词，不发送 S1/S2/S3 业务数据。
5. 点击“保存设置”，然后重新运行 `启动Steven便携Demo.cmd`。
6. API Key 使用 Windows DPAPI 按当前 Windows 用户加密，保存于本机 `data\secrets`；不会写入数据库、日志、README 或使用手册。
7. 如需恢复离线模式，重新打开配置窗口并取消勾选“启用在线 DeepSeek”，保存后重启。

交付包预填 DeepSeek Endpoint 和模型，但不会内置真实测试 Key。把真实 Key 放进可复制 ZIP 会使任何取得压缩包的人都能使用该凭据；每台演示电脑必须由负责人单独填写并保存。更换电脑或 Windows 用户后需要重新配置。

## 5. 停止与验证

- 正常停止：双击 `停止Steven便携Demo.cmd`。重复停止是安全的。
- 环境自检：启动后双击 `验证Steven便携Demo.cmd`。
- 自检项目包括：Session 模式、PostgreSQL 持久化、PaddleOCR Provider、Alembic `20260718_0016`、两个账号、HTTPS 页面与便携 Python 依赖。

## 6. 首页、通知、历史与设置

- Dashboard：查看 S1 文书、S2 采购比价、S3 库存盘点入口。
- 通知中心：查看审批和退回通知，可逐条或全部标记已读并跳转到业务详情。
- 历史记录：聚合 S1/S2/S3 操作，可按模块筛选并查看对象详情。
- 个人设置：切换繁体/简体界面、修改显示名称和密码。修改密码需要验证旧密码。
- 系统固定文案可切换繁简；用户输入、SKU、金额、文件名、Word/Excel 内容和数据库数据不会自动转换。

## 7. S1 标书／文书模块

### Steven 操作

1. 打开“标书／文书”。
2. 新建文书或使用扫描导入新增文书。
3. 选择至少一种脱敏模板，填写标题、文书编号、日期、预算、地点、供应商和受控条款。
4. 在实时预览区检查模板效果。
5. 如使用图片/PDF，上传后由 PaddleOCR 离线识别；不确定字段进入人工确认，不会直接写入正式业务数据。
6. 检查未替换 `{{变量}}`、日期、预算和重复供应商规则。
7. 可执行 AI 校对演示，逐条接受或忽略候选建议；默认 AI 为本地 Mock。
8. 人工修订后提交审批。

### approve 操作

1. 使用 approve 登录。
2. 打开已提交文书。
3. 选择批准或退回；退回必须填写意见。

### 导出

- 仅已批准文书可导出正式 Word。
- 每次导出生成新版本，历史版本不覆盖。
- 支持选择多个供应商，批量生成相互独立的 Word。
- 已批准文书可使用“打印全部”，每个供应商单独分页。

## 8. S2 采购比价模块

### Steven 操作

1. 打开“采购比价”，新建报价事项。
2. 可采用结构化录入、Excel 导入或扫描报价单新增数据。
3. 扫描流程为：图片/PDF上传 → PaddleOCR → AI候选 → 表格式人工复核 → 确认写入。
4. 可一次识别多个供应商；必须检查供应商、品项、数量、单价、有效期和币种。
5. 系统计算合计、规则异常和比较结果；人工决定推荐供应商。
6. 历史供应商搜索只读取已批准或已导出的报价，可选择是否复用供应商及其历史商品信息。
7. 可生成询价函草稿用于复制，但系统不会实际发送。
8. 提交审批。

### approve 操作

1. 打开已提交比价事项。
2. 核对人工推荐与依据。
3. 批准或填写意见退回。

### 导出

- 仅已批准事项可生成正式 Excel。
- 每次导出追加新版本，历史版本不覆盖。
- AI 推荐只供参考，不会自动选择供应商、下单或审批。

## 9. S3 消耗品库存模块

### Steven 操作

1. 打开“消耗品库存”，查看标准脱敏库存品项。
2. 可创建/编辑品项，维护 SKU、品项、类别、库位、账面量、安全库存和目标库存。
3. 可使用固定格式 Excel 导入、DeepSeek 智能列映射界面或自然语言快速录入界面；便携包默认 AI Mock，因此需人工确认映射和结果。
4. 发起盘点，录入盘点量。
5. 系统自动计算差异、低库存标记与建议订货量：`max(0, 目标库存 - 盘点量)`。
6. 人工确认补货量；与系统建议不同时必须填写理由。
7. 提交审批。

### approve 操作

1. 打开已提交盘点单。
2. 核对差异、低库存、系统建议和人工理由。
3. 批准或填写意见退回。

### 导出

- 仅已批准盘点单可导出正式 Excel，支持 v1/v2/v3 等追加版本。
- 库存首页可导出全部库存信息 Excel。

## 10. OCR 与 AI 说明

- PaddleOCR：真实离线运行，模型位于包内 `apps\api\models\ocr`，支持 PNG、JPG、PDF。
- OCR 结果：必须经过人工确认，不能直接决定预算、供应商、推荐、审批或正式库存值。
- AI：便携包默认 `mock`，不会在未配置时访问公网；使用 `配置Steven大模型API.cmd` 可显式切换到 DeepSeek。
- 在线模式只允许使用者在目标电脑本机填写 Key；Key 由 Windows DPAPI 保护，配置窗口和日志均不回显。
- DeepSeek 输出仍必须进入人工确认流程，不得自动决定供应商、预算、推荐、审批、下单或正式业务值。

## 11. 常见问题

### 端口被占用

默认端口为 PostgreSQL 55432、FastAPI 9000、Next.js 4300、HTTPS 15443。先停止占用程序，或由技术人员通过 PowerShell 脚本参数指定替代端口。

### HTTPS 证书警告

先启动一次，再运行 `信任Steven本地HTTPS证书.cmd`。该操作只信任本包当前生成的本机 Caddy CA，需要管理员确认。

### 启动超时

检查 `data\runtime\logs` 中的日志。不要把日志中的运行环境信息发到公开渠道；日志设计上不写数据库密码、Cookie 或 Token。

### 中文路径与临时虚拟盘

当交付包位于中文路径时，初始化和启动脚本会自动选择空闲盘符（例如 `S:`）并通过 Windows `subst` 建立临时 ASCII 路径。Demo 运行期间不要手动断开该盘符；执行 `停止Steven便携Demo.cmd` 后会自动释放。若所有候选盘符均被占用，先释放一个未使用盘符后重试。

### OCR 首次较慢

PaddleOCR 首次加载模型需要较多时间和内存。请等待，不要连续重复提交。

### 忘记停止

双击停止脚本。脚本只停止本包记录的进程和本包 PostgreSQL 数据目录，不停止电脑上的其他 PostgreSQL 服务。

### 换 Windows 用户后无法启动数据库

数据库密码使用 Windows DPAPI，绑定首次初始化的 Windows 用户与电脑。若复制到另一台电脑或切换用户，必须使用未初始化的原始 ZIP 重新解压并执行首次初始化，不能复制已初始化后的 `data\postgresql` 和 `data\secrets`。

## 12. 演示边界

Steven S1/S2/S3 通过，仅代表本机、独立、脱敏 Demo 的完整业务闭环可演示；不代表生产可用、受控试运行、已连接在线 OCR/AI，或已获得进一步上线授权。演示账号密码仅用于隔离的本机脱敏 Demo，禁止复用于其他系统。
