# Open Design Brief：Steven S2 采购比价

为香港培英中学校务处 Steven 专属行政助手设计 `/dashboard/steven/quotes` 采购比价页面。延续 Phase 1 Dashboard 的克制、可信、行政工作台视觉，不生成其他岗位入口。

页面必须覆盖采购事项列表、5 个标准品项、3 家供应商、15 条报价明细、导入预检、异常阻断、最低价排序、人工推荐、审计和 Excel 版本导出。明确区分系统价格排序与人工推荐：推荐字段默认空白，不自动选择供应商。

核心交互：

- PageHeader 唯一主操作“新建采购事项”。
- 列表显示事项、币种、供应商数、品项数、完整性、有效期和推荐状态。
- 详情页使用品项表、供应商报价卡和 3×5 报价矩阵。
- 导入、异常、附件、审计、版本放入抽屉或弹窗。
- 导入预检必须显示行号、字段、严重程度，并由人工确认后写入。
- 14 条缺报价、币种不一致时阻断完整比较；过期报价醒目标红。
- 运费和税费为供应商级固定金额，每家只计一次。
- 最低价排序仅为系统计算结果；人工推荐供应商默认空白。
- 推荐非最低价时，非最低价理由和审批意见必须同时填写。
- 导出前显式确认；每次 Excel 导出创建新版本，不覆盖历史。
- AI/OCR 禁用；不得出现自动推荐、自动审批、下单、发信或供应商搜索能力。

请输出：

1. 可直接预览并具备列表、详情、导入抽屉、异常抽屉、推荐弹窗、版本抽屉、确认和 Toast 示例交互的完整 HTML 原型。
2. Next.js + TypeScript 组件拆分、状态、props、响应式和无障碍说明。
3. 使用 Phase 1 色彩基线：背景 #F6F7F9、卡片 #FFFFFF、边框 #D8DEE6、正文 #1D2939、弱文本 #667085、危险 #B42318、成功 #027A48、警告 #936A00。
